Scientific EDA via Python

Pandas / Seaborn / Matplotlib

I utilized Pandas to engineer 50k+ rows, performing complex lambda transformations and handling outlier detection with statistical precision.

Using Seaborn & Matplotlib, I uncovered deep-seated correlations between order velocity and shipping modes, creating heatmaps that revealed critical logistics bottlenecks.

SQL Data Warehousing

The cleaned data was moved to a structured SQL environment to ensure ACID compliance and high-performance reporting.

-- Sample Query: Aggregating Performance
SELECT region, SUM(sales) AS total_revenue
FROM ecommerce_master_table
GROUP BY region
ORDER BY total_revenue DESC;
